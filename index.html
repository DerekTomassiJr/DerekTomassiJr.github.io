In this project I created a bot for Discord using the DiscordPy API. The bot was made to create events for servers. A user would create an event and give it a title, user count, time and date, and optionally a game to associate with the event. After the event was created users could use commands in the text channel to join the event if the event hadn't already occured or if the event wasn't full. At the time of the event the bot would ping joined users that the event has started and a voice channel to join.
